What is difference between relative and absolute URL ?

ans :- Relative url ->Partial path which is download in our computer .


Example of Relative URL :
               <img src="images/photo.jpg" alt="Example Image" >

ans :- Absolute URL:-absolute url is full path of the image works anywhere globally


        Example of absolute URL :
        < img src="https://www.example.com/images/photo.jpg" alt="Example Image"
